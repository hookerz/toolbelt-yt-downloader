How To Use

Paste a YouTube Video ID into the ID field and click Get Info. The application will download the avaliable audio and video files for this video. You can test with this ID YE7VzlLtp-4

After you provide a valid ID you need to choose which folder the files will be saved to and which files you want to save. You probably want the ones with combined audio and video but you can choose as many as you like and the application will download them all.
